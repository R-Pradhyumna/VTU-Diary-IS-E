VTU INTERNSHIP DIARY GENERATOR (IS&E VERSION) - COMPLETE STEPS

===========================================
STEP 1: EXTRACT DATA FROM VTU PORTAL
===========================================

1. Open the VTU diary page:
   https://vtu.internyet.in/dashboard/student/diary-entries

2. Make sure you are logged in.

3. Open Developer Tools:
   - Press F12
   - Go to the "Console" tab

4. Open the file "fetchData.js" from this project.

5. Copy the entire contents of fetchData.js.

6. Paste it into the browser console and press Enter.

7. Wait for the script to finish:
   - You will see logs like "Fetching page X..."

8. After completion, a file will be downloaded:
   refined_diary.json

-------------------------------------------
IMPORTANT:
- Do NOT close the tab while the script is running
- Ensure all pages are fetched before proceeding
-------------------------------------------


===========================================
STEP 2: GENERATE SKILLS USING AI (REQUIRED)
===========================================

WARNING:
If you skip this step, the generated diary will be incomplete.

The extracted JSON does NOT contain skills.  
You must generate skills for each entry before uploading.

OPTION 1 (Manual):

1. Open refined_diary.json

2. For each entry, use AI (e.g., ChatGPT) with prompt:

   "Generate 1–2 relevant technical skills for this work summary:
    <paste work_summary>"

3. Add a new field to each entry:

   "skills": "Skill1, Skill2"


OPTION 2 (RECOMMENDED - Faster):

1. Copy the entire JSON file

2. Paste it into ChatGPT

3. Ask:

   "Add 1–2 technical skills to each entry and return the full JSON"

4. Copy the result


Example:

{
  "date": "2026-02-02",
  "hours": 6,
  "skills": "Problem Analysis, Technical Assessment",
  "work_summary": "...",
  "learnings": "..."
}

5. Save the updated file as:
   diary_with_skills.json

-------------------------------------------
IMPORTANT:
- Do NOT modify existing fields
- Only ADD the "skills" field
- Keep skills short and technical (1–2 per entry)
-------------------------------------------


===========================================
STEP 3: GENERATE DIARY (UPLOAD JSON)
===========================================

1. Open the file:
   index.html

2. In the "Upload JSON File" section:
   - Click "Choose File"
   - Select "diary_with_skills.json"

3. Click:
   "Generate Diary"

4. You will be redirected to:
   viewer.html


===========================================
STEP 4: SAVE AS PDF
===========================================

1. On the viewer page, click:
   "Print / Save as PDF"

   OR press:
   Ctrl + P

2. In the print settings:

   - Destination → Save as PDF
   - Pages → All
   - Layout → Portrait
   - Margins → Default (or Custom if needed)

3. VERY IMPORTANT:
   - Disable "Headers and Footers"

   (This removes date, URL, and extra text from the PDF)

4. Click:
   Save

5. Choose location and save the PDF.


===========================================
FINAL OUTPUT
===========================================

You will get a clean IS&E-formatted internship diary PDF with:

- One diary entry per page
- Clearly structured sections:
  - Skills (AI-generated)
  - Description
  - Learnings/Outcomes
- Bullet formatting for improved readability
- Proper spacing and justified text
- Signature sections included on every page


===========================================
NOTES
===========================================

- Recommended browser: Google Chrome
- Do NOT refresh viewer.html after upload (data will be lost)
- Data is stored in localStorage
- Keep both JSON files safe for reuse
- Ensure print scale is set to 100% for correct formatting
